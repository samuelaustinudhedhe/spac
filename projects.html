Python Problem-Solving Toolkit

Two small Python programs from my COS102 (Problem Solving) lab assessment: a right-angled triangle checker that validates and analyzes triangle angles, and a palindrome number checker that reverses digits with a while loop. I also built an extended version that draws the triangle visually using math and matplotlib.pyplot.

Laravel Livewire Select Component

An open-source x-select component I built and maintain for Laravel Livewire. It fixes a real bug where Livewire's DOM morphing doesn't reliably preserve the selected attribute on native <select> elements, so dropdowns stay in sync after a Livewire re-render.
